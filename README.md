## 🧠 Sobre o Projeto

O **Flash Bootcamp** é um treinamento intensivo da **NearX**, projetado para capacitar novos desenvolvedores a compreenderem e aplicarem conceitos essenciais da **Web3** — com o poder da **IA** acelerando o processo de desenvolvimento.

**O que você vai aprender:**
- 🪙 Fundamentos e estrutura do **Blockchain**
- 📜 Introdução aos **Smart Contracts**
- 🤖 Aplicações práticas de **Inteligência Artificial no Front-end**
- 🧩 Ferramentas e fluxos modernos de produtividade

## 🏗️ Estrutura do Projeto

```bash
📦 flash-bootcamp-web3 <br>
 ┣ 📂 src <br>
 ┃ ┣ 📂 components     <br>  # Componentes reutilizáveis (React)
 ┃ ┣ 📂 contracts   <br>     # Smart Contracts do projeto
 ┃ ┣ 📂 pages      <br>      # Páginas principais da aplicação
 ┃ ┗ 📜 main.js     <br>     # Ponto de entrada do Front-end
 ┣ 📜 package.json <br>
 ┣ 📜 README.md <br>
 ┗ 📜 .env.example <br>

⚙️ Como Executar o Projeto

# Clone o repositório
git clone https://github.com/seu-usuario/flash-bootcamp-web3.git

# Acesse o diretório
cd flash-bootcamp-web3

# Instale as dependências
npm install

# Execute o projeto
npm run dev
